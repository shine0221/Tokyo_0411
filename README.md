# 🐰🦊 東京七日旅行協作手帳

這是使用 Vue 3、Tailwind CSS 和 Firebase Realtime Database 建立的旅行計畫與記帳應用程式。

## 功能簡述
* 行程規劃與天氣預報
* 行李與預訂清單
* 共同記帳與結算

📝 分頁功能詳情
應用程式分為四個主要分頁，可在底部導航欄切換：

1. 行程 (Itinerary)
多日行程： 支援 7 天 的行程規劃，可點擊日期標籤快速切換。
行程列表： 依時間排序顯示每日行程，包含時間、地點、備註。
預訂追蹤： 可標記行程是否「需預訂」，並可切換「已完成預訂」狀態。
天氣預報： 顯示當天的東京天氣預報，包含最高/最低溫、天氣描述與建議。
地圖助手： 提供一個「導航助手」清單，彙整所有行程地點並提供 Google 地圖導航連結。

2. 準備 (Preparation)
進度概覽： 顯示 預訂進度 和 行李打包進度 的百分比。
預訂清單： 將行程中標記「需預訂」的項目彙整成清單，可勾選完成。
行李清單： 提供可新增、刪除、勾選的打包清單，並可匯入常用清單。

3. 翻譯 (Translate)
旅行會話： 內建基礎、超商、餐廳、購物等分類的中日文常用短語。
語音播放： 點擊短語可使用 SpeechSynthesisUtterance 進行 日文發音。

4. 記帳 (Expenses)
記帳分頁包含「明細」與「記帳」兩種模式。

4.1. 明細模式 (Details)
即時換算: 快速將日幣金額轉換為台幣估價。
總結/預算切換卡片 (v12.0):
總結/結算模式: 顯示以新台幣 (TWD) 計算的總支出金額，以及 兔子、狐狸個別支付的總金額 (TWD)。
日幣預算模式: 顯示「兔子🐰」和「狐狸🦊」各自的日幣現金預算使用進度。
明細列表： 可按日期群組顯示所有支出明細，可依付款人（兔子、狐狸、全體）篩選。
共同支出結算： 自動計算兩人的總共同支出、各自代墊金額 (以 JPY 計)，並得出最終結算金額 (誰應付誰，並顯示 TWD 總額)。

4.2. 記帳模式 (Add)
提供表單，包含日期、金額、幣別 (JPY/TWD)、項目、付款人、類別 (飲食、交通、購物、娛樂、其他)、付款方式 (現金、信用卡、西瓜卡)、是否為共同支出等欄位。

⚙️ 技術細節與 API
前端框架： 使用 Vue 3 的 Composition API (setup 函數) 進行狀態管理和邏輯控制。
資料儲存： 使用 Firebase Realtime Database 作為資料後端，資料儲存在 /tokyo_trip_data 節點下。
同步狀態： 實作了同步狀態指示器 (sync-status)，顯示「已同步」、「儲存中...」或「連線中斷」。
權限控制： 透過密碼鎖定編輯模式，確保資料安全。

外部 API 來源：
匯率： 呼叫 https://api.exchangerate-api.com 取得 JPY 到 TWD 的即時匯率。
天氣： 呼叫 https://api.open-meteo.com 取得東京的預報資訊。
