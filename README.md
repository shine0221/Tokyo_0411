# 🐰🦊 東京七日旅行協作手帳

這是使用 Vue 3、Tailwind CSS 和 Firebase Realtime Database 建立的旅行計畫與記帳應用程式。

## 功能簡述
* 行程規劃與天氣預報
* 行李與預訂清單
* 共同記帳與結算
  
📝 應用程式分為四個主要分頁，可在底部導航欄切換：

1. 行程 (Itinerary)
多日行程： 支援 7 天 的行程規劃。
行程列表： 依時間排序顯示每日行程，包含時間、地點、備註。
預訂追蹤： 可標記行程是否「需預訂」或「已完成預訂」。
天氣預報： 顯示當天的 東京天氣預報，包含最高/最低溫、天氣描述與建議。
地圖助手： 提供一個「導航助手」清單，列出所有行程地點並提供導航連結。

2. 準備 (Preparation)
進度概覽： 顯示 預訂進度 和 行李打包進度 的百分比。
預訂清單： 將行程中標記「需預訂」的項目彙整成清單，可勾選完成。
行李清單： 提供可新增、刪除、勾選的打包清單，並可匯入常用清單。

4. 翻譯 (Translate)
旅行會話： 內建基礎、超商、餐廳、購物等分類的中日文常用短語。
語音播放： 點擊短語可使用 SpeechSynthesisUtterance 進行 日文發音 (speak(phrase.ja))。

4. 記帳 (Expenses)
總支出： 顯示以 新台幣 (TWD) 計算的總支出金額，使用即時匯率換算。
現金預算： 顯示「兔子🐰」和「狐狸🦊」各自的日幣現金預算使用進度。
明細列表： 可按日期群組顯示所有支出明細，可依付款人（兔子、狐狸、全體）篩選。
共同支出結算： 自動計算兩人共同支出的總額、各自代墊金額，並得出最終結算金額 (誰應付誰，並顯示 TWD 總額)。
記帳表單： 包含日期、金額、幣別 (JPY/TWD)、項目、付款人、類別 (飲食、交通等)、付款方式 (現金、信用卡、西瓜卡)、是否為共同支出等欄位。

☁️ 資料同步與技術細節
雲端同步： 使用 Firebase Realtime Database 作為資料後端，確保「兔子🐰」和「狐狸🦊」的資料能即時同步。
資料儲存在 Firebase 的 tokyo_trip_data 節點下。
應用程式實作了同步狀態指示器 (sync-status)，顯示「已同步」、「儲存中...」或「連線中斷」。

匯率與天氣 API：
匯率： 呼叫 https://api.exchangerate-api.com 取得 JPY 到 TWD 的即時匯率。
天氣： 呼叫 https://api.open-meteo.com 取得東京的預報資訊。
前端框架： 使用 Vue 3 的 Composition API (setup 函數) 進行狀態管理和邏輯控制。
